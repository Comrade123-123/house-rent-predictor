### House Rent Prediction

## Overview
This is my first end-to-end Machine Learning project, built to predict house rental prices using a real-world dataset.

The project covers the complete machine learning workflow, including:
- Data loading
- Exploratory Data Analysis (EDA)
- Data cleaning and preprocessing
- Feature engineering
- Categorical encoding
- Model training
- Model evaluation

## Project Objective
The goal of this project was to build a regression model capable of predicting house rent based on various property features such as:
- BHK
- Size
- City
- Furnishing Status
- Bathroom
- Tenant Preference
- Area Type
- Point of Contact

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## Machine Learning Pipeline
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Encoding Categorical Features
6. Train-Test Split
7. Linear Regression Model
8. Model Evaluation

## Results
A Multiple Linear Regression model was used as the baseline model.

The model achieved limited predictive performance because the relationship between the available features and the target variable (Rent) is not completely linear. Additionally, the dataset contains significant variability that cannot be captured effectively by a simple linear model.
